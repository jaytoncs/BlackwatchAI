# 🛡️ Blackwatch AI

> AI-powered Cyber Threat Intelligence & Decision Support Platform

Blackwatch AI is an AI-powered cybersecurity platform that aggregates live threat intelligence from trusted sources and transforms it into actionable, executive-ready insights.

Instead of forcing analysts to sift through hundreds of vulnerabilities and security advisories, Blackwatch AI prioritizes emerging threats, highlights actively exploited vulnerabilities, and provides AI-assisted recommendations to support faster security decisions.

---

## 🚀 Current Features

- Live National Vulnerability Database (NVD) integration
- Live CISA Known Exploited Vulnerabilities (KEV) integration
- Interactive Streamlit dashboard
- Mission Control threat overview
- Executive threat summaries
